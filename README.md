# 一梳梳到头

"一梳梳到头"是一款线上售卖梳子的应用。

  *购买方式*
  1.选择商品-商品详情-加入购物车,在购物车进行批量下单,不限种类不限数量;
  2.选择商品-商品详情-立即下单,进行单件商品的结算。

  *产品特色*
  1.随时更新商品类型和种类,以及时下流行风格;
  2.商家不定时发布优惠活动,优惠券等;
  3.手机立即下单,商品到付;
  4.通过手机号码可直接注册成为我们用户;
  5.两种购买方式任意挑选;
  6.商品详情提供该商品详细信息。
